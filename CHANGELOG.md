# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release
- Material Design UI with dark mode support
- Multi-user authentication (Google OAuth, Email/Password)
- Role-based access control (Admin, User, Guest)
- Task CRUD operations with priorities, labels, deadlines, attachments
- Drag & drop task reordering
- Real-time sync with Firebase Firestore
- Offline mode with AsyncStorage
- Interactive dashboard with charts
- Push notifications for reminders and deadlines
- Calendar views (Month, Week, Day)
- Recurring tasks functionality
- Collaboration features (task sharing, team chat, comments, activity log)
- Import/Export (Excel, CSV, PDF)
- Bulk edit operations
- Advanced search and filtering
- Multilingual support (English, Romanian)
- Biometric authentication
- Encrypted local storage
- Settings for theme, notifications, and integrations
- Onboarding tour
- CI/CD pipeline with GitHub Actions
- Comprehensive documentation

### Security
- Biometric lock support
- Encrypted storage for sensitive data
- Secure authentication flow
- Role-based permissions
